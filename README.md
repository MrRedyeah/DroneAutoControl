# DroneAutoControl
我构建了一套基于大语言模型与强化学习的无人机飞控代码自动化迭代 Agent，解决嵌入式飞控开发中传统调试依赖人工、周期长、极端场景覆盖不全的核心痛点。 核心逻辑流采用多模块协作闭环：1）代码解析 Agent 自动提取飞控工程中传感器驱动、姿态控制逻辑的关键代码段，基于项目规范完成语法与架构校验；2）仿真测试 Agent 调用 MATLAB/Simulink 飞控模型，自动生成 100 + 极端工况测试用例（如强风干扰、GPS 信号丢失），驱动仿真并收集姿态误差、控制延迟数据；3）优化推理 Agent 基于测试结果与强化学习奖励函数，自动生成控制参数调优建议与代码优化补丁，完成闭环迭代。
